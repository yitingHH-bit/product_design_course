# Kotlin Higher-Order Functions Practice

This project demonstrates the use of Kotlin's higher-order functions: `filter`, `map`, and `reduce`.

## Features
- Creates a list of integers from 1 to 100
- Filters even numbers
- Maps all numbers to their squares
- Reduces the list to the sum of all elements
- Prints results with clear labels
