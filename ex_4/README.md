# Product Banner App (Kotlin Compose)

This Android app displays a simple **product banner** using **Jetpack Compose**.
The banner includes:

* A full-screen **product image** as the background
* **Product name**
* **Company name**
* **Contact information**
* Proper layout with alignment and spacing for a clean UI

## Features

* Built with **Jetpack Compose**
* Background image loaded from `res/drawable`
* Semi-transparent overlay for readable text
* Fully responsive layout that adapts to screen size

## Files Included

* **MainActivity.kt** – main application code with the Compose UI
* **drawable/** – contains background product image (`image1.jpg`)
* **screenshot.png** – screenshot of the running app
* **README.md** – program description (this file)

## How to Run

1. Clone or download the project
2. Open in **Android Studio**
3. Run on an emulator or physical device
